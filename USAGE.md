# 使用指南

## 快速上手

### 第一步：配置 API Key

1. 访问 [DeepSeek 平台](https://platform.deepseek.com/) 注册账号
2. 获取 API Key
3. 在项目根目录创建 `.env` 文件：

```env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxx
DEEPSEEK_API_BASE=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-chat
```

### 第二步：启动系统

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

### 第三步：访问系统

打开浏览器访问：http://localhost:8000

## 四诊信息详细说明

### 望诊

观察患者的：
- **面色**：红润、苍白、萎黄、晦暗、潮红等
- **舌象**：
  - 舌质：淡红、淡白、红、绛、紫暗等
  - 舌苔：薄白、厚白、黄、黄腻、少苔、无苔等
  - 舌形：胖大、瘦小、有齿痕、有裂纹等
  - 舌上有无瘀斑、瘀点
- **形体**：胖瘦、姿势、步态
- **神色**：精神状态、眼神

### 闻诊

注意：
- **声音**：语声高亢、低微、嘶哑等
- **呼吸**：气息强弱、喘促等
- **咳嗽**：咳声、痰声
- **口气**：有无异味

### 问诊（十问歌）

1. **寒热**：怕冷还是怕热，有无发热
2. **汗**：出汗情况（自汗、盗汗、无汗等）
3. **头身**：头痛、头晕、身体疼痛等
4. **胸腹**：胸闷、腹胀、疼痛等
5. **饮食**：食欲、口味、饮水量
6. **二便**：
   - 大便：次数、性状（干、稀、成形）
   - 小便：次数、颜色、量
7. **睡眠**：入睡情况、睡眠质量、多梦等
8. **疼痛**：部位、性质、时间
9. **月经**（女性）：周期、经期、经量、颜色、痛经等
10. **情志**：情绪状态、压力情况

### 切诊

脉象特征：
- **浮沉**：浮脉、沉脉、中脉
- **迟数**：迟脉（<60次/分）、数脉（>90次/分）
- **虚实**：有力、无力
- **其他**：滑、涩、弦、细、洪、弱、濡、紧等

常见脉象：
- 浮脉：轻取即得，表证
- 沉脉：重按始得，里证
- 迟脉：脉来缓慢，寒证
- 数脉：脉来急数，热证
- 滑脉：脉来流利，痰湿
- 涩脉：脉来艰涩，血瘀
- 弦脉：脉体紧张，肝病
- 细脉：脉细如线，血虚

## 诊疗流程说明

1. **提交信息** → 系统验证输入
2. **中医辨证** → 八纲、六经、脏腑等辨证（约10-15秒）
3. **文献分析** → 并行查询古籍和现代文献（约20-30秒）
4. **处方开具** → 经方大师根据分析结果开方（约15-20秒）
5. **安全审方** → 并行进行用药安全审核和康复方案（约15-20秒）
6. **生成报告** → 整合所有结果生成完整报告（约10秒）

总耗时：约 **70-100 秒**

## 结果解读

### 中医辨证结果

包含：
- 八纲辨证（阴阳表里寒热虚实）
- 六经辨证（太阳阳明少阳太阴少阴厥阴）
- 卫气营血辨证（外感热病）
- 脏腑辨证（心肝脾肺肾）
- 病机总结

### 处方说明

- **主方**：选用的核心经方
- **药物组成**：每味药的剂量（克）
- **煎服法**：如何煎煮服用
- **中成药**：推荐的医保中成药
- **疗程**：建议服用时间

### 审方结果

重点关注：
- **配伍禁忌**：有无十八反十九畏
- **特殊人群**：妊娠、肝肾功能等
- **剂量审核**：是否符合药典标准
- **安全性评级**：安全/需调整/有风险

### 康复方案

包含：
- **针灸**：主穴配穴、手法
- **推拿**：部位手法
- **食疗**：食疗方、饮食宜忌
- **气功**：功法导引
- **生活调护**：起居情志运动

## 常见问题 FAQ

### Q1: API 调用失败怎么办？

**A:** 
1. 检查网络连接
2. 验证 API Key 是否正确
3. 确认 DeepSeek 账户余额充足
4. 查看控制台错误信息

### Q2: 诊疗结果不准确怎么办？

**A:**
- AI 结果仅供参考，不能替代医生诊断
- 可以多次尝试，调整输入信息的详细程度
- 四诊信息越详细，结果越准确
- 建议咨询专业中医师

### Q3: 可以直接按处方用药吗？

**A:**
- **不可以！** 本系统仅供参考学习
- 必须由有资质的中医师诊断后开方
- 切勿自行用药

### Q4: 报告可以保存吗？

**A:**
- 可以导出 Markdown 格式
- 可以打印或截图保存
- 可以复制到其他文档

### Q5: 如何提高诊疗质量？

**A:**
1. 提供尽可能详细的四诊信息
2. 准确描述症状的时间、程度、诱因
3. 说明既往病史和正在服用的药物
4. 特殊情况（妊娠、过敏等）务必注明

### Q6: 费用如何计算？

**A:**
- DeepSeek API 按 tokens 计费
- 单次诊疗约调用 7 次 API
- 预计单次费用：0.5-2 元（根据输入长度）
- 建议在 DeepSeek 平台设置用量限制

## 最佳实践

### ✅ 推荐做法

1. **详细输入**：提供完整的四诊信息
2. **准确描述**：症状描述具体准确

5. **完全依赖**：不要完全依赖 AI 诊断
