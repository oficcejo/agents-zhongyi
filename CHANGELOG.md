# 更新日志

本文档记录项目的所有重要变更。

## [1.0.0] - 2024-11-05

### ✨ 新功能

#### 核心功能
- 🤖 实现 7 个专业 AI 智能体协同诊疗系统
  - Agent 1: 中医辨证（八纲+六经+卫气营血+脏腑）
  - Agent 2: 古籍病案分析（汉唐至民国医案）
  - Agent 3: 现代文献分析（CNKI 核心期刊）
  - Agent 4: 经方大师开方（含剂量换算）
  - Agent 5: 药剂师审方（安全性审核）
  - Agent 6: 康复理疗方案（针灸推拿食疗气功）
  - Agent 7: 报告生成（完整诊疗报告）
